此页面列出了内核和核心插件公开的 KV 惯例。
每个条目定义了键所代表的信息以及当前发布该键的组件。
有关 KV 操作和存储行为，请参阅 [KV存储指南](./ipc-kv)。

## 内核插件记录

内核在只读的 `sys` 命名空间中发布插件元数据。
这些记录使其他组件能够区分可用的插件包和已向内核注册的插件实例。

| 键模式 | 定义者 | 含义 |
| --- | --- | --- |
| `sys/plugin/catalog/<插件名称>` | 内核 | 扫描期间发现的插件包的元数据。 |
| `sys/plugin/<实例 ID>` |内核 | 已加载插件实例及其注册资源的元数据。 |

目录记录描述包元数据，包括插件标识、版本、后端、合约版本、命令、包路径和清单元数据。
它并不指示插件是否正在运行。

运行实例记录描述已注册的实例及其资源，包括路由、静态挂载和 Socket.IO 命名空间。
当实例停止或卸载时，内核会移除此记录。降级后的实例仍然可以拥有记录；该记录反映了已成功注册的资源。


## 已发布的应用程序元数据

插件在从拥有该资源的插件派生的命名空间下发布元数据。
以下键是其他插件或前端代码可能需要发现的资源的当前惯例。


| 键 | 定义者 | 值含义 |
| --- | --- | --- |
| `web-assets/urls` | `web-assets` 插件 |一个 JSON 对象，将每个已发布的 Web 资源名称映射到其公共 URL。
| `web-assets/style` | `web-assets` 插件 | 共享样式表的公共 URL。|
| `web-assets/color-scheme` | `web-assets` 插件 | 颜色方案样式表的公共 URL。|
| `web-assets/icon` | `web-assets` 插件 | 共享图标的公共 URL 前缀。|
| `web-assets/culori` | `web-assets` 插件 | Culori JavaScript 资源的公共 URL。|
| `web-sdk/urls` | `web-sdk` 插件 | 一个 JSON 对象，将每个 SDK 资源名称映射到其公共 URL。|
| `web-sdk/sdk` | `web-sdk` 插件 | Web SDK 脚本的公共 URL。|
| `web-sdk/languages` | `web-sdk` 插件 | SDK 语言元数据的公共 URL。|

聚合 `urls` 键及其对应的单个键描述的是同一资源集。

发现所有资源时使用聚合键；需要特定资源时使用单个键。
