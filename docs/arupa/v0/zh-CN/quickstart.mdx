Arupa 是一款基于插件的服务器管理工具。您只需运行一个内核，然后添加

插件即可获得所需的管理功能。

## 内核

Arupa 内核是主机进程。它提供 HTTP 和 Socket.IO

服务器，加载插件，并应用共享的身份验证和访问规则。

内核本身不提供完整的管理界面。

## 插件

插件是为内核添加功能的软件包。插件可以提供

HTTP 路由、静态文件和 Socket.IO 命名空间。核心插件提供 Arupa 安装的基本组件；您可以根据需要添加更多插件。

## 后续操作

1. 使用二进制文件或 [Docker 安装内核](./kernel/installation)。

2. 从[Coreplugins](../../../coreplugins/v0/en/index)获取所需的软件包。

3. [安装插件包](./kernel/plugins#install-a-plugin)并配置

用户和访问规则，然后再将 Arupa 提供给其他人使用。
