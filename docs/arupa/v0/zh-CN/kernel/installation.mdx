---
title: "安装 Arupa"
description: "安装 Arupa 内核，然后添加所需的核心插件。"
---

Arupa 由两部分组成：内核和插件。内核提供

服务器运行时环境。插件则添加在其上运行的管理功能。

首先安装内核。然后获取所需的核心插件并将其添加到

内核安装中。

## 1. 安装内核

选择适合您环境的安装方法：

<CardGroup cols={2}>

<Card title="从二进制文件安装" icon="file-binary" href="./installation-binary">

在 Linux 或 macOS 上运行已发布的内核二进制文件。

</Card>

<Card title="使用 Docker 安装" icon="docker" href="./installation-docker">

在 Docker 容器中运行内核。

</Card>

</CardGroup>

## 2. 获取核心插件

内核安装完成后，选择并获取提供所需功能的插件。

首先请参阅[核心插件文档](../../../../coreplugins/v0/zh-CN/introduction)。

要将下载的插件包添加到正在运行的内核，请参阅[安装插件](./plugins#install-a-plugin)。
